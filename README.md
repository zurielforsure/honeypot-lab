# Hybrid Honeypot Lab User Manual

## Overview
Complete deployment and configuration guide for a hybrid honeypot lab using:
- **Cowrie** - SSH honeypot
- **Splunk** - Log visualisation and alerting
- **Suricata** - IDS for malicious traffic detection
- **Hydra** - Brute-force attack simulation

## What's Inside
This repository contains the **user manual** (PDF) documenting:
- Cowrie honeypot setup and configuration
- Splunk log ingestion and dashboard creation
- Suricata IDS deployment and testing
- Hydra brute-force attack simulation
- Real-time alerting and log analysis workflow
